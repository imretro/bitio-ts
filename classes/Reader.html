Class Reader

Reads bits from a Uint* typed array.

The number of bits per number depends on the size of the Uint. For example, a Uint16Array results in 16 bits being read for each number in the array.

Bits are read from largest to smallest. For example, with new Uint16Array([1, 1 << 15]), 1 is the last bit read from the first element, and the first bit read from the second element. This order is left-to-right, assuming big-endian numbers.

This acts as an iterator, meaning it is not possible to reverse a read.

    Use with readBitsSafe and .filter to safely collect bits into a number. The most significant bit will be on the left, so [1, 1, 0, 1] would equal the numeric literal 0b1101.
